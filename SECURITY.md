# Security policy

Report vulnerabilities through GitHub's private
[security advisory form](https://github.com/pawnkit/pawn-parser/security/advisories/new).
Please do not open a public issue before a fix or mitigation is available.

The parser handles untrusted source. Crashes, hangs, excessive memory use, and
invalid source ranges caused by a small input are in scope. Include the shortest
input that reproduces the problem when possible.
